## Python

```python
import redis
import traceback

_redis = redis.StrictRedis(host='1.2.3.4', port=2233, db=0)
with _redis.pipeline() as pipe:
  pipe.setex('key', 60*2, 'value')
  try:
    pipe.execute()
  except Exception as e:
    traceback.print_exc()
```
